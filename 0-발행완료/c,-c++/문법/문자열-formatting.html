문자열 Formatting

C 스타일 with printf

출력형식 뜻 설명 출력형식 뜻 설명
%d Decimal 부호있는 10진수 정수 %ld Long Decimal 부호 있는 long 형 10진수 정수
%lld Long Long Decimal 부호 있는 long long 형 10진수 정수
%u Unsigned 부호 없는 10진수 정수 %lu Long Unsinged 부호 없는 long 형 10진수 정수
%llu Long Long Unsigned 부호 없는 long long 형 10진수 정수
%p Pointer 주소 (16진수)
%x heXadecimal 부호 없는 16진수 정수 (소문자) %X heXadecimal 부호 없는 16진수 정수 (대문자)
%o Octal 부호 없는 8진수 정수
%c Character 문자 하나 %s String 문자열. 널문자(\0)를 만날 때까지
%f Float 부호 있는 float형 실수 (유효자리 6) %lf Long Float 부호 있는 double형 실수 (유효자리 15)
정렬형식 설명 정렬형식 설명
%{숫자}{출력형식} {숫자} 만큼 공백문자를 채워 정렬 %0{숫자}{출력형식} {숫자} 만큼 0으로 채워 정렬
%.{숫자}f(lf) (실수형 출력에서) {숫자} 만큼만 소수점 이하 자리 출력 
printf("%08.2f", 3.141592f); // 00003.14

C++ 스타일

<iomanip> (Input Output Manipulator) with cout

출력형식 설명 출력형식 설명
dec 10진수 정수 hex 16진수 정수
oct 8진수 정수 showbase 0x(16진수), 0(8진수) 접두어 표기
uppercase 16진수 문자 대문자로 표기 fixed 고정 소수점 표기
setprecision(n) 소수점 이하 (n-1) 자리만 출력 setw(n) 길이 n만큼 공백문자를 채워 정렬
setfill(ch) 정렬할 경우 공백 대신 ch 문자로 채움
left 좌측 정렬 right 우측 정렬
internal 중앙 정렬 
#include <iomanip>

cout << setprecision(3) << setw(8) << setfill('0') << 3.141592f; // 00003.14

std::format (C++20 이후)

포맷 설명 포맷 설명
{} 기본 출력 {:d} 정수형 출력
{:x} 16진수 소문자 출력 {:X} 16진수 대문자 출력
{:.nf} 소수점 n자리까지 출력
{:>10} 폭 10, 오른쪽 정렬 {:<10} 폭 10, 왼쪽 정렬
{:^10} 폭 10, 중앙 정렬 {:08d} 폭 8, 0으로 채움, 정수형 
cout << std::format("{:>10}", 123) << "\n";   // "       123"
cout << std::format("{:^10}", "hi") << "\n";  // "   hi    "
cout << std::format("{:08X}", 255) << "\n";   // 000000FF
